# Asset Catalog

## Description
Lists the SVG/PNG assets that are part of the visual design (icons, decorative circles, background illustrations).

## Assets
| Asset name | File (store in `src/design_system/assets/`) | Usage |
|------------|--------------------------------------------|-------|
| `hero‑background.png` | `hero-background.png` | Hero section background |
| `circle‑decor.svg` | `circle-decor.svg` | Decorative circles used in USPs, Features, Services |
| `logo‑grid.svg` | `logo-grid.svg` | Partner‑logo placeholder (replace with actual partner logos) |
| `cta‑illustration.svg` | `cta-illustration.svg` | Optional illustration inside CTA block |
| `service‑illustration‑left.svg` | `service-illustration-left.svg` | Image for Services (image‑right layout) |
| `service‑illustration‑right.svg` | `service-illustration-right.svg` | Image for Services 2 (image‑left layout) |

> **Tip:** Keep all assets version‑controlled in the `src/design_system/assets/` folder; the developer can replace the placeholder SVGs with the final exported assets from Figma.
